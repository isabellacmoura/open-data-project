Sobre o Projeto

O projeto "Dados Abertos em Portugal" é uma iniciativa que incentiva o uso de dados abertos na cobertura de políticas públicas em Portugal.

Objetivos

Este projeto tem como principais objetivos:

  • Promover a transparência e acesso a dados públicos
  • Mapear a disponibilidade de dados nos municípios portugueses
  • Organizar eventos e workshops sobre dados abertos
  • Identificar datasets de interesse público que não estão disponíveis

Estrutura do Projeto

O projeto está organizado em três pastas principais:

  • Pasta Vazia: Lista de datasets de interesse público que não estão disponíveis
  • Pasta Protegida: Mapeamento de como cada município disponibiliza dados
  • Pasta Partilhada: Hackatons e workshops para obter dados públicos

Contribuições
