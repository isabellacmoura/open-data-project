Open Data Municipalities Project

This project emerged from the need to promote and map open data and the use of the Law on Access to Administrative Documents (LADA) in Portugal. The goal is to create a platform that facilitates citizen and journalist access to municipal information, promoting transparency and local data journalism.

The project is organized into three main components: RAI Database (research and data collection), Organized Municipalities (geographic data), and Web Application (interactive platform). This represents a significant effort to democratize access to public information in Portugal, facilitating the work of journalists, researchers, and citizens interested in municipal data and administrative transparency.
