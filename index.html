Dados Abertos em Portugal
Projeto que incentiva o uso de dados abertos na
cobertura de políticas públicas em Portugal
Sobre o projeto / Referências / Contacto
Em pasta vazia reunimos uma lista de datasets que apesar de serem de interesse público não estão disponíveis. Em pasta protegida mapeamos como cada município de Portugal disponibiliza dados. Em pasta partilhada organizamos e divulgamos hackatons e workshops para obter dados públicos.
